PRÒLOGO


Esta publicación llamada el diario de un programador, apuntes de seguridad, Python desde cero, el autor aborda el aprendizaje de Python de una forma dinámica buscando que el usuario final tome el aprendizaje de una forma sencilla a través de ejemplos y situaciones cotidianas que le van a permitir abordar la programación de una forma natural.

Este libro más que un manual técnico pretende ser un acercamiento a la programación desde una visión diferente, pero también es el camino para conocer la importancia de Python en la seguridad informática, la cual es enorme, especialmente hoy en día. La programación no comienza con un lenguaje, ni con una computadora encendida frente a nosotros. Comienza con una pregunta.

Este diario, El diario de un programador: apuntes de seguridad: apuntes de seguridad, Python desde cero, no es solo una guía técnica, ni una colección de códigos para memorizar. Es el registró de un camino: el de alguien que decidió entender cómo funcionan las cosas detrás de la pantalla, como se construyen, pero también como se protegen, y como a veces se rompen.

Un diario no se escribe en un solo día. Está lleno de errores, descubrimientos, dudas y pequeños logros que, con el tiempo, se convierten en conocimiento real. Así es también el aprendizaje de la programación y a seguridad informática. No es inmediato, no es perfecto, pero si profundamente transformador.

Aquí encontraras más que instrucciones: encontraras apuntes. Ideas que nacen desde cero, explicaciones pensadas para quien comienza, y una mirada hacia la seguridad informática no como algo lejano o exclusivo de expertos, sino como una habilidad que puede desarrollarse paso a paso. Python será nuestra herramienta, pero no nuestro limite. A través de el aprenderemos a pensar, a resolver problemas y a comprender mejor el mundo digital en que vivimos. Este diario no pretende darte todas las respuestas. Pretende enseñarte a hacer mejores peguntas.
